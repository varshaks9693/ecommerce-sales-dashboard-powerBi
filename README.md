# 📊 E-commerce Sales Dashboard – Madhav Store

## 📌 Project Overview
This project analyzes e-commerce sales data of Madhav Store using Power BI.
The goal is to identify key business insights such as sales trends, customer behavior, and profitability.

## 🛠️ Tools & Technologies

* Power BI
* Data Cleaning
* Data Visualization

## 📈 Dashboard Features

* Sum of Amount by State (Sales Distribution)
* Sum of Quantity by Category
* Sum of Amount by Customer Name
* Payment Mode Analysis (Category-wise)
* Monthly Profit Analysis
* Profit by Sub-Category

-## 🔍 Key Insights

* Identified top customers contributing highest revenue
* Found most popular product categories based on quantity sold
* Analyzed preferred payment modes of customers
* Observed monthly profit trends to identify growth/decline
* Highlighted high-profit and low-profit sub-categories
